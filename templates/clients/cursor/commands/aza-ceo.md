# /aza-ceo — Product / CEO review

Activate the **ceo** role from AzaLoop DynamicBinder.

1. Call `aza_session(action=calibrate)` if needed
2. Call `aza_prd(action=review, title=..., description=...)` for new work
3. Challenge: user, wedge, non-goals, smallest shippable slice
4. Prefer sharper PRD over feature pile-on
5. When ready: `aza_prd(action=approve)` then `aza_loop(action=full)`
