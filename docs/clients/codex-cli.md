# Codex CLI (OpenAI) (`codex-cli`)

> Tier1: ⚪ experimental | Category: cli | Since: 0.1.2

## 环境变量

- `AZA_CLIENT_NAME=codex-cli`
- `OPENAI_API_KEY`

## 安装步骤

- npm i -g @openai/codex
- 配置 ~/.codex/config.toml

## 启动命令

```bash
codex --mcp-server aza
```

## 推荐工具

- `aza_session`
- `aza_auto`
- `aza_loop`

## 已知限制

- CLI 模式
