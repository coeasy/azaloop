# Goose (Block) (`goose`)

> Tier1: ⚪ experimental | Category: cli | Since: 0.1.2

## 环境变量

- `AZA_CLIENT_NAME=goose`

## 安装步骤

- pip install goose-ai
- 配置 ~/.config/goose/config.yaml

## 启动命令

```bash
goose session --with-mcp aza
```

## 推荐工具

- `aza_session`
- `aza_auto`
- `aza_loop`

## 已知限制

- CLI 模式
