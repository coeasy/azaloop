# Trae (`trae`)

> Tier1: ✅ | Category: ide | Since: 0.1.0

## 环境变量

- `AZA_CLIENT_NAME=trae`
- `AZA_CLIENT_VERSION`

## 安装步骤

- 1. 安装 Trae (trae.ai)
- 2. pnpm add -g @azaloop/mcp-server
- 3. 在 Trae AI 助手设置 → MCP Server 添加
- 4. 配置 aza mcp serve --client trae

## 启动命令

```bash
aza mcp serve --client trae
```

## 推荐工具

- `aza_session`
- `aza_prd`
- `aza_auto`
- `aza_loop`
- `aza_meta`

## 已知限制

- 部分 next_action 需 ai run 显式调用
