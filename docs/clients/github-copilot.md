# GitHub Copilot (`github-copilot`)

> Tier1: ⚪ experimental | Category: ide | Since: 0.1.2

## 环境变量

- `AZA_CLIENT_NAME=github-copilot`
- `GH_TOKEN`

## 安装步骤

- VSCode + Copilot Chat 扩展 + 启用 MCP 协议

## 启动命令

```bash
copilot --mcp-server aza
```

## 推荐工具

- `aza_session`
- `aza_loop`

## 已知限制

- MCP 协议需 Copilot Chat ≥ 0.18
