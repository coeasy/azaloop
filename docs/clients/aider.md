# AzaLoop × Aider

**Tier:** T3 Minimal | **MCP:** ❌ | **Hooks:** ❌ | **Auto-Loop:** ❌

## 快速开始

```bash
npx @azaloop/cli init --client aider
```

## 配置

| 文件 | 位置 |
|------|------|
| 规则文件 | `CONVENTIONS.md` |

Aider 不支持 MCP 协议，AzaLoop 通过 `CONVENTIONS.md` 规则文件提供指导。

## 使用

在 Aider 中，规则文件自动加载为系统提示的一部分。

## Troubleshooting

- Aider 无 MCP 支持，需手动跟踪循环
- 降级模式：手动执行 PRD → 实现 → 验证
