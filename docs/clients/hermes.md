# Hermes (Nous Research) (`hermes`)

> Tier1: ⚪ experimental | Category: cli | Since: 0.1.2

## 环境变量

- `AZA_CLIENT_NAME=hermes`
- `NOUS_API_KEY`

## 安装步骤

- pip install hermes-agent
- 配置 ~/.hermes/mcp.json

## 启动命令

```bash
hermes --mcp-server aza
```

## 推荐工具

- `aza_session`
- `aza_loop`

## 已知限制

- CLI 模式
