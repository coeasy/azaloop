# Qwen Code (通义灵码) (`qwen-code`)

> Tier1: ⚪ experimental | Category: ide | Since: 0.1.2

## 环境变量

- `AZA_CLIENT_NAME=qwen-code`
- `DASHSCOPE_API_KEY`

## 安装步骤

- VSCode/JetBrains 安装 Lingma 扩展
- 切换 Qwen Code 模式

## 启动命令

```bash
qwen-code --mcp-server aza
```

## 推荐工具

- `aza_session`
- `aza_loop`

## 已知限制

- 国内 token 走 DashScope
